# A-Better-NYC
# Why Is It So Hard to Find Affordable Housing in NYC?

## Project Overview
This project analyzes affordable housing in New York City using two NYC Open Data datasets. The goal is to understand how affordable housing is distributed across the five boroughs and compare housing production with Housing Connect lottery opportunities.

## Datasets
- [Advertised Lotteries on Housing Connect by Building](https://data.cityofnewyork.us/Housing-Development/Advertised-Lotteries-on-Housing-Connect-By-Buildin/nibs-na6y/about_data)

- [Affordable Housing Production by Building](https://data.cityofnewyork.us/Housing-Development/Affordable-Housing-Production-by-Building/hg8x-zxpr/about_data)

- [Local Law 44 Unit Income Rent](https://data.cityofnewyork.us/Housing-Development/Local-Law-44-Unit-Income-Rent/9ay9-xkek/about_data)

## Tools Used

## Project Steps
- Imported and explored the datasets.
- Cleaned the data by handling missing values and standardizing borough names.
- Analyzed affordable housing across NYC boroughs.
- Created visualizations to compare housing production and lottery buildings.

7. Available Data in Local Law 44 Dataset

## Key Findings
- Affordable housing is not evenly distributed across NYC.
- The Bronx has the largest share of affordable housing units.
- Brooklyn has many Housing Connect lottery buildings.
- Most affordable housing units are intended for low-income households.
